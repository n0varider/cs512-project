# cs512-project

This was a project for CS512: Computer Vision. The project implements a spectral vision transformer from the SpectFormer paper. The paper can be found [here](https://openaccess.thecvf.com/content/WACV2025/papers/Patro_SpectFormer_Frequency_and_Attention_is_What_You_Need_in_a_WACV_2025_paper.pdf "here").

To compensate for a lack of computing resources and data, we used [CutMix](https://openaccess.thecvf.com/content_ICCV_2019/papers/Yun_CutMix_Regularization_Strategy_to_Train_Strong_Classifiers_With_Localizable_Features_ICCV_2019_paper.pdf "CutMix") and [Mixup](https://arxiv.org/pdf/1710.09412 "Mixup") to help the model generalize better.
